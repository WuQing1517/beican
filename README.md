# 备餐系统

智能营养配比系统，通过线性规划算法自动计算最优食品配比。

## 功能特性

- **营养物质管理**：自定义添加蛋白质、碳水化合物、脂肪等营养物质
- **每日需求配置**：设置每种营养物质的每日需求范围
- **餐数配置**：支持3餐、4餐等多种餐数方案
- **每餐约束设置**：为每餐设置营养物质的最小/最大值约束
- **食品管理**：添加食品及其每100g的营养成分
- **智能配比生成**：自动生成满足所有约束条件的食品配比方案

## 快速开始

### 1. 安装依赖

```bash
cd D:\meal-planner
python -m pip install -r requirements.txt
```

### 2. 启动应用

```bash
python app.py
```

### 3. 访问系统

打开浏览器访问：http://localhost:5000

## 使用流程

1. **配置营养物质** → 进入"营养物质"页面，添加或编辑营养物质
2. **设置每日需求** → 进入"每日需求"页面，设置每种营养物质的每日摄入范围
3. **配置餐数方案** → 进入"餐数配置"页面，创建3餐、4餐等方案
4. **添加食品** → 进入"食品管理"页面，添加常用食品及其营养成分
5. **生成配比** → 进入"生成配比"页面，选择餐数方案和食品，点击生成

## 数据存储

所有数据存储在 `data/` 目录下的JSON文件中：

- `nutrients.json` - 营养物质定义
- `daily_requirements.json` - 每日营养需求
- `meals.json` - 餐数配置方案
- `foods.json` - 食品营养数据
- `plans.json` - 生成的历史配比方案

## 技术栈

- Python 3.x
- Flask 3.0
- PuLP (线性规划求解器)
- Bootstrap 5 (前端UI)
