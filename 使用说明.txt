====================================
        备餐系统使用说明
====================================

【功能介绍】
本系统是一个智能营养配比系统，帮助您科学规划每一餐。

主要功能：
- 营养物质管理
- 每日营养需求配置
- 约束规则配置
- 餐数配置（支持固定餐）
- 食品管理
- 智能配比生成
- 每日许愿功能
- 抽奖转盘彩蛋

【快速开始】

1. 安装Python
   - 下载地址: https://www.python.org/downloads/
   - 安装时请勾选"Add Python to PATH"

2. 启动系统
   - 双击"启动.bat"文件
   - 等待依赖安装完成
   - 打开浏览器访问: http://localhost:5000

3. 或者手动启动
   - 打开命令行
   - 进入项目目录
   - 执行: pip install -r requirements.txt
   - 执行: python app.py
