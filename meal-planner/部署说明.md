# 部署到Render云平台

## 步骤1: 注册Render账号
1. 访问 https://render.com/
2. 点击 "Get Started" 注册账号
3. 可以使用GitHub账号登录

## 步骤2: 推送代码到GitHub
1. 在GitHub创建新仓库
2. 将项目代码推送到仓库

```bash
cd D:\meal-planner
git init
git add .
git commit -m "初始提交"
git remote add origin https://github.com/你的用户名/meal-planner.git
git push -u origin main
```

## 步骤3: 在Render创建Web Service
1. 登录Render控制台
2. 点击 "New" -> "Web Service"
3. 连接你的GitHub仓库
4. 配置设置：
   - Name: meal-planner
   - Runtime: Python
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python app.py`
   - Plan: Free

## 步骤4: 等待部署
Render会自动构建和部署，大约需要2-3分钟。

## 步骤5: 访问应用
部署完成后，Render会提供一个URL，格式类似：
`https://meal-planner-xxxx.onrender.com`

## 注意事项
1. 免费版有冷启动限制，长时间不访问后首次访问会较慢
2. 数据存储在文件系统中，重启后数据会丢失
3. 如需持久化数据，建议使用数据库
